## Оценка COCOMO (Basic, органический)

| Параметр | Значение |
|----------|----------|
| Объём кодовой базы | ~10 000 SLOC |
| Формула | E = 2.4 × (KLOC)^1.05 |
| Расчёт E | 2.4 × 10^1.05 ≈ **27 чел.-мес.** (теоретически) |
| Факт | ~2 чел.-мес. одного разработчика за 8 недель |
| Ускорение | Spring Boot, Expo, готовые библиотеки |

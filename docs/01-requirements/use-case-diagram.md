# Use Case диаграмма

![Use Case](../images/fig-02-use-case.png)

Рисунок 2 — Диаграмма вариантов использования

## Актор

**Пользователь мобильного приложения** — игрок LoL или зритель статистики.

## Варианты использования

| ID | Название | Описание |
|----|----------|----------|
| UC-001 | Поиск и профиль | Поиск по Riot ID, просмотр ранга, матчей, союзников |
| UC-002 | Избранное | Добавление/удаление игроков по PUUID |
| UC-003 | Регистрация и вход | Создание учётной записи, получение JWT |
| UC-004 | Привязка Riot ID | Связь локального пользователя с игровым аккаунтом |
| UC-005 | Детали матча | KDA, чемпионы, участники |

## Связи

- UC-001 `include` UC-005 (просмотр матчей из профиля)
- UC-002 `include` UC-001 (открытие профиля из избранного)
- UC-003 → UC-002, UC-004 (аутентификация для избранного)

# SWOT-анализ

## Сильные стороны

- Официальная интеграция с Riot Games API
- Кэширование в PostgreSQL снижает нагрузку на внешний API
- Мобильный клиент — доступ с Android-устройства
- Архитектура PCMEF упрощает тестирование и сопровождение
- 16 REST-эндпоинтов (требование траектории В: 8+)

## Слабые стороны

- Зависимость от лимитов и доступности Riot API
- Development API Key действует ~24 часа
- Нет push-уведомлений об изменении ранга
- Ограниченный офлайн-режим (только кэш последнего профиля)

## Возможности

- Статистика по чемпионам, сравнение игроков
- iOS-сборка, публикация в магазинах приложений
- Production Riot API Key

## Угрозы

- Изменение политики Riot в отношении сторонних приложений
- Конкуренция с OP.GG, u.gg, Porofessor
- Rate limit при росте числа пользователей

# Бизнес-глоссарий (этап 0)

Полный глоссарий (26+ терминов): см. также [../01-requirements/glossary-extended.md](../01-requirements/glossary-extended.md).

Ключевые термины предметной области:

| Термин | Определение |
|--------|-------------|
| Призыватель (Summoner) | Игрок LoL; объект поиска и отображения статистики |
| Riot ID | Отображаемое имя в формате Имя#Тег; используется для поиска |
| PUUID | Постоянный идентификатор игрока; ключ для матчей и избранного |
| Матч (Match) | Завершённая игровая сессия с уникальным matchId |
| Избранное | Список призывателей, сохранённых пользователем приложения |
| Cache-Aside | Стратегия кэширования: сначала БД, при устаревании — Riot API |

Исходная версия: [../01-business-model/glossary.md](../01-business-model/glossary.md) (архив).

# Руководство пользователя

## 1. Регистрация и вход

1. Откройте приложение LoL Companion
2. Нажмите «Нет аккаунта? Зарегистрироваться»
3. Введите логин, email и пароль
4. После регистрации войдите с теми же данными

## 2. Поиск призывателя

1. Перейдите на вкладку **Поиск**
2. Выберите **регион** (RU, EUW, NA и др.)
3. Введите **Riot ID** в формате `Имя#Тег`
4. Нажмите **Найти**
5. Откроется экран профиля

## 3. Просмотр профиля

На экране профиля отображаются:
- уровень, ранг, LP, винрейт;
- последние матчи (нажмите для деталей);
- частые союзники;
- кнопка **В избранное**.

## 4. Избранное

1. Вкладка **Избранное** — список сохранённых игроков
2. Нажмите на карточку — откроется профиль
3. Кнопка **×** — удалить из избранного

## 5. Привязка своего Riot ID

1. Вкладка **Аккаунт**
2. Привяжите Riot ID (Имя#Тег + регион)
3. Кнопка **Мой профиль** на экране поиска откроет ваш профиль

## 6. Выход

**Аккаунт** → «Выйти и войти в другой аккаунт»

## Ошибки

| Сообщение | Действие |
|-----------|----------|
| Призыватель не найден | Проверьте Riot ID и регион |
| Сервер недоступен | Запустите backend |
| Rate limit | Подождите и повторите |

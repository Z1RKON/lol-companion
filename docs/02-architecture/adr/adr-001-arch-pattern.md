# ADR-001: Выбор архитектурного паттерна PCMEF

**Статус:** Принято

## Контекст

Курсовой проект требует слоевой архитектуры с чётким разделением ответственности. Траектория В предполагает мобильный клиент и Java backend.

## Решение

Enterprise-адаптация PCMEF:
- Presentation — React Native (Expo)
- Control, Mediator, Entity, Foundation — Spring Boot
- Интеграция через REST/JSON + JWT

## Последствия

- (+) Единая бизнес-логика на сервере
- (+) Клиент остаётся тонким
- (−) Сетевая зависимость mobile от backend
